Node.js的版本升级

2022-07-16 09:36:47
  • javascript
  • nodejs

# 查看本地node版本

node -v

# 安装n模块

npm install -g n（mac需要在命令的前面加上sudo）

# 升级到指定的版本/最新版本，升级之前，可以执行n ls（查看可以升级的版本）

n 8.10.1（版本号）
或者你可以安装最新的稳定版本
n stable

# 4.安装完成以后，检查一下是否升级成功

node -v
查看本地node版本
